# nodejs-toolbox
Sample node.js Package to demonstrate javascript as command line scripting language

Start a node.js application

```
npm init
npm install --save-dev grunt-contrib-concat

```


Create Gruntfile.js

Running grunt to perform task

```
grunt concat
grunt concat:deploy
```
